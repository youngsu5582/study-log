# AWS 한글 문서 찾는팁

가끔씩 AWS 문서가 영문만 있거나

![](https://i.imgur.com/11HMF0M.png)

와 같이 언어로 번역이 되어 있지 않다고 번역 요청 하라고 하는 경우도 있다.

이때 `https://docs.aws.amazon.com/s3/` 가 링크라면
-> `https://docs.aws.amazon.com/ko_kr/s3/` 와 같이 입력해보자.

![](https://i.imgur.com/UnrhruY.png)

한글로 번역된 문서가 나타난다!

없는 경우도 있겠지만, 생각보다 많은 문서가 한글로 번역되어 있다.
